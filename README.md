# snake_nn

This is a frontend-only demo that trains neural networks to play Snake using an evolutionary strategy (selection + mutation).

How to run

1. Open `index.html` in a browser (live server recommended).
2. Use the controls to choose inputs, number of layers/neurons, population size and mutation rate.
3. Click Start to run generations. The best agent's weights and activations are shown live.

Files
- [index.html](index.html) — UI
- [style.css](style.css) — styles
- [app.js](app.js) — neural net, game, evolver and UI code

This is intentionally simple and runs entirely in the browser.
